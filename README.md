🚖 Boalkhali Go

Boalkhali Go is a modern ride-sharing platform built with HTML, CSS, JavaScript and Firebase.

The platform supports three user roles:

- Rider
- Driver
- Admin

---

Features

Rider

- User Registration
- User Login
- Book Ride
- View Active Ride
- Trip History
- Profile Dashboard

---

Driver

- Driver Registration
- Vehicle Information
- License Verification
- NID Verification
- Profile Photo
- Online / Offline Mode
- Accept Ride
- Complete Ride
- Earnings Dashboard

---

Admin

- Secure Admin Access
- Driver Approval System
- Driver Rejection System
- View Riders
- View Drivers
- Dashboard Statistics
- Platform Analytics

---
